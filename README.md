# Left - 温馨时光记录

一个温馨的个人时光记录项目，包含计时器、情话分享、育儿知识等功能。

## 功能特性

- 💝 优雅的计时器显示
- 🌟 动态灵动岛提示
- 💫 流畅的交互动画
- 🎨 精美的视觉设计
- 📱 完善的响应式支持

## 项目结构

```
left/
├── index.html          # 首页
├── about.html          # 关于页面
├── joke.html           # 趣味空间
├── parenting.html      # 育儿知识
├── fortune.html        # 运势查询
├── common.css          # 公共样式
└── README.md          # 项目说明
```

## 技术栈

- HTML5
- CSS3
- JavaScript
- Lunar Calendar API
- 温馨提示 API
